About the wealth index

The index shown here is a composite score designed to compare asset wealth across census tracts and racial/ethnic groups in Maryland over time. The index is meant to be complementary to other indicators of wealth available from the Black Wealth Data Center. It is built from U.S. Census American Community Survey data using indicators that reflect household wealth, such as:

  • Homeownership rate
  • Vehicle ownership (proxied by commute)
  • SNAP receipt (as total assets are a qualifying factor)
  • Household crowding (occupants per room)
  • Types of housing units (e.g., detached homes)

These indicators are combined using principal component analysis (PCA) to create a single index value using methodology from USAID. Higher values indicate greater access to wealth-related resources and "gap" indicators show the difference between groups.

Note: The index is relative and not a direct measure of dollars or assets. Tracts with estimated populations below 50 have been omitted. Full replication code is available here.
